## 项目介绍

本项目用于：批量下载指定且已获得请求同意网站的 m3u8 视频，监测并移除视频中的广告 TS 文件

注意：项目 **无法直接运行** ，因为 config.py 已被加入 .gitignore。将缺失 URL 和 cookie
